## Getting Started

### Dependencies
* Python 3.7
* Pytorch 1.6.0
* Sklearn

the environment can be created by utilizing MFGR.yml
```
conda env create -f MFGR.yml
```
## Run the Code
We employ CIFAR-100 as an example
```
sh run.sh
```
